# Guia do Processo de Dockerização

### Clonagem do Repositório

Para realizar a clonagem do repositório, crie uma nova pasta, abra um terminal na raiz da pasta e digite o seguinte código:

```
git clone https://github.com/JuaoLuicas/dockerizacao-pratica.git
```

### Execução do Docker
Após realizar a clonagem, execute o seguinte comando:

```
docker-compose up
```
Com o comando executado, as imagens das aplicações serão criadas e executadas em seguida.

### Exclusão dos Containers
Para realizar a exclusão dos containers, utilize o seguinte comando:

```
docker-compose down
```
